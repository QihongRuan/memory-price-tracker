## Notes on 10-Year Charts

The 10-year historical charts in this project use:
- **Verified data points** from above sources where available
- **Interpolated estimates** between verified points
- **Market pattern simulation** for periods without verified data

All simulated data is clearly labeled and should not be used for trading decisions.
